# @medisphere/common-ui

Shared Svelte UI library for MediSphere frontend projects.

## Install

```sh
npm install @medisphere/common-ui
```

## Use

```ts
import { cn } from '@medisphere/common-ui';
```

## Development

```sh
npm run dev
npm run check
npm run build
```

## Package Output

Library artifacts are generated in `dist/` via `svelte-package`.
